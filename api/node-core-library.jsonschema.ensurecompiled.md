## JsonSchema.ensureCompiled() method

If not already done, this loads the schema from disk and compiles it.

<b>Signature:</b>

```typescript
ensureCompiled(): void;
```
<b>Returns:</b>

`void`

## Remarks

Any dependencies will be compiled as well.
